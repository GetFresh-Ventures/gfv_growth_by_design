# Outreach Cadence Workflow

**When**: Launching outreach to new prospects
**Duration**: Initial design: 30 min | Per-touch execution: 5-10 min
**Skills used**: [outreach-sequence](../skills/outreach-sequence/SKILL.md), [email-composer](../skills/email-composer/SKILL.md)

---

## Steps

### 1. Qualify the Prospect (5 min)
Before building a sequence, verify:
- [ ] This person/company is a genuine fit for what we do
- [ ] We have a specific reason to reach out NOW (trigger event)
- [ ] We have a warm path or relevant angle (not pure cold outreach)
- [ ] They're not already in our CRM pipeline

### 2. Design the Sequence (15 min)
```
Build a 5-touch outreach sequence for [Name] at [Company].
Context: [why now, warm path, angle].
Use the outreach-sequence skill.
```

### 3. Review All Touches
Before sending anything:
- [ ] Each touch is genuinely personalized
- [ ] Escalating value (not repeating the same pitch)
- [ ] Voice model compliant (reads like you wrote it)
- [ ] Multi-channel where possible (email + LinkedIn)
- [ ] Has a graceful close (Touch 5 isn't desperate)

### 4. Execute Touch by Touch

**Day 0 — Touch 1: Opening email**
```
Draft Touch 1 for [Prospect]. Use email-composer skill.
Reference: [specific trigger/reason for outreach]
```
→ Wait for approval → Send → Log in CRM

**Day 3-5 — Touch 2: LinkedIn**
- Connect with personalized note OR
- Comment on their recent post (authentic engagement)
→ Log in CRM

**Day 7-10 — Touch 3: Value-add email**
```
Draft Touch 3 for [Prospect]. Share [specific insight/resource].
Keep it brief. No ask yet.
```
→ Wait for approval → Send → Log in CRM

**Day 14-18 — Touch 4: Soft ask**
```
Draft Touch 4 for [Prospect]. Make a specific ask:
"15 minutes Thursday?" not "let's find time."
```
→ Wait for approval → Send → Log in CRM

**Day 25-30 — Touch 5: Graceful close**
```
Draft Touch 5 for [Prospect]. Acknowledge they're busy.
Leave the door open. No guilt.
```
→ Wait for approval → Send → Log in CRM

### 5. Evaluate & Learn
After the sequence completes (or they respond):
```
Log the outreach outcome in brain/learnings.md:
- What worked? What didn't?
- Did they respond? At which touch?
- Would you change the approach?
```

---

## Cadence Rules
- **Never skip the approval step** for any touch
- **Stop the sequence immediately** if they respond (switch to conversation)
- **Don't compress timing** — spacing matters for not feeling spammy
- **Adapt** — if you learn something new mid-sequence, adjust remaining touches
- **One sequence per prospect** — don't run parallel campaigns to the same person
